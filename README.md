Mix-Tape

![Mix-Tape](https://raw.github.com/brandtabbott/mix-tape/master/public/mixtape.gif)
================
Make a mix tape for your loved one!!  I love mix tapes.

Installation
------------

Execute:

    $ npm install
    $ npm start

Goto URL:

    http://localhost:3000
